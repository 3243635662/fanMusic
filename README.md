# fanMusic (Cool Demo) 学习ing

这只是一个简单的 Nuxt 4 练习 Demo，仿了一下 Apple Music 界面，练手用的。

## 🕹️ 随便玩玩

- **API 来源**: [KuGouMusicApi](https://github.com/MakcRe/KuGouMusicApi)
- **技术栈**: Nuxt 4, Nuxt UI, Pinia, Tailwind CSS
- **功能点**:
  - 搜索歌曲、看推荐
  - 基础的音乐播放、切歌、队列管理
  - 简单的换肤和状态保存
